# Éléments obsolètes / supprimés

Ce document liste les systèmes retirés du projet pour éviter les confusions.

## Supprimés

- **MP / Mana** – Plus de points de mana. Magie via baguettes/consommables uniquement.
- **Affixes** – Système d'affixes (data/affixes.lua, data/cursed_affixes.lua) supprimé. Objets créés à la main.
- **blockCast** – Action "Lancer un sort" retirée. Les effets mutisme/peur ne bloquent plus de cast.
- **mind_burn** – Effet drain PM supprimé avec les MP.
- **potion_mp** – Consommable retiré.

## Fichiers supprimés

- `data/affixes.lua`
- `data/cursed_affixes.lua`
- `core/affixes/affix_registry.lua`

## Documentation obsolète

- `ANALYSE_AFFIXES_MAUDITS.md` – Décrit l'ancien système d'affixes maudits.
- `docs/GUIDE_AJOUT_DATA.md` – Sections 4 (Affixes), mentions heal_mp, mpCost à ignorer.
- `CHARACTER_SYSTEM.txt` – Référence MP.
- `EQUIPMENT_SYSTEM.txt` – Référence affixes.
